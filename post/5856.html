Javascript 函数语法

函数是由事件驱动的或者当它被调用时执行的可重复使用的代码块。

函数包裹在花括号的代码块，前面使用函数名funciton

Javascript 对大小写敏感。关键词function必须事小写的，并且必须以与函数名称相同的大小写来调用函数。

调用带参数的函数

在调用函数时，可以向其传递值，这些值被称为参数。

例:myfunction(argumentl,argument2)
变量和参数必须按一致顺序出现。第一个变量对应的是第一个被传递的参数的值。
