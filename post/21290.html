# Mysql 基础语法

  • Insert 语句：向某一行插入一条数据。
    在查询编辑器中输入以下 sql.

INSERT INTO user
( mobile , name , sex )
VALUES
(‘123’,‘张三’,‘1’) // 这里不是反单引号！

  • Delete 删除语句：

DELETE FROM
表名
WHERE 值 = 表内对应值

  • Select 语句：

SELECT *
表名
FROM 表名 // 查询表内全部内容

  • Update 语句：

UPDATE user
SET name = ’值‘
WHERE mobile = ’值‘ // 根据 mobile 来确认修改数据的值

# 远程连接 MySQL 数据库的一些问题

记录报错 2002（10060）

  1. 服务器安全组需要开放数据库的端口（默认 3306）。
  2. 假如使用宝塔面板则需要在安全内放行端口。
  3. 若上方内容还是不行则在服务器命令窗口输入关闭防火墙的命令。
