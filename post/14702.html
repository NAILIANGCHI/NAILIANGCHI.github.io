# 循环语句

while 语句 给定它一个值的范围等于，不等于或者大小于它都会一直运行直至超出或者小于给定值停止运行。

for 语句
Python for 循环可以遍历任何序列的项目，如一个列表或者一个字符串
可以循环字符串

结果挨个换行打印出 china

for i in range(5)
含义是遍历 0-5 前一个结束会从 0 开始计数至数字 4 结束

若想从一计数那么需要加上参数
for i in range (1,5)
含有则是从 1 开始到 5 前一个数字结束
会输出 1-4

for 语句里面还能包含三个值例如：
for i in range(1, 10, 3)
for i in range (起始值， 结束值， 步长)
1 开始步长为 3
1+3=4
4+3=7
7+3=10 // 但是只能 1-9 所以 10 不能输出
则会打印 1 4 7
** 应用场景：** 会爬取一个列表返回给我们
循环一个列表
例如：
a_list = [' 周杰伦 '，' 林俊杰 ', ' 成龙 ']
for i in a_list:
print(i)
break 语句
break 语句用在 while 和 for 循环中。
作用：停止执行后面的循环语句

continue 语句
Python continue 语句跳出本次循环，而 break 跳出整个循环。
continue 语句用来告诉 Python 跳过当前循环的剩余语句，然后继续进行下一轮循环。
continue 语句用在 while 和 for 循环中。

pass 语句
作用：是空函数，不做任何事情，用于暂时还未写逻辑和函数的里面

Python 日期和时间
用函数方法
time.time ( ) 来获取当前时间戳
格式化时间
print(time.strftime("%Y-%m-%d %H:%M:%S", time.localtime()))
