Python基础

发表于2022-10-19|更新于2022-11-07|python
|阅读量:

变量

直接赋值例如：
a = 1
b = 2
注意：字符串赋值需加引号“”如 c = "刘强"

数据类型

整数类型：int
字符串类型：str
浮点数：float
等等..

数组

注意取单个字符需要从0开始计数例如
‘’a,b,c,d”–>0,1,2,3
若[:5]–>则是从1开始计数

字典

var = {‘name’: ‘张三’,’sex’:’男’,’age’: ‘20’}

函数
常用的Python内助的函数
input( ) 用于接收用户输入的数据
print( ) 用于打印数据

判断语句

假若
a = 3
b = 2
如果a和b相等则会执行上面的语句反之执行下一行的
if 判断条件：
执行语句……
else：
执行语句……
注意：缩进需注意

循环语句

while 语句 给定它一个值的范围等于，不等于或者大小于它都会一直运行直至超出或者小于给定值停止运行。

for 语句
Python for循环可以遍历任何序列的项目，如一个列表或者一个字符串
可以循环字符串

结果挨个换行打印出china

for i in range(5)
含义是遍历0-5前一个结束会从0开始计数至数字4结束

若想从一计数那么需要加上参数
for i in range(1,5)
含有则是从1开始到5前一个数字结束
会输出1-4

for语句里面还能包含三个值例如：
for i in range(1, 10, 3)
for i in range(起始值， 结束值， 步长)
1开始步长为3
1+3=4
4+3=7
7+3=10 //但是只能1-9所以10不能输出
则会打印1 4 7
应用场景：会爬取一个列表返回给我们
循环一个列表
例如：
a_list = [‘周杰伦’，’林俊杰’, ‘成龙’]
for i in a_list:
print(i)
break 语句
break语句用在while和for循环中。
作用：停止执行后面的循环语句

continue 语句
Python continue 语句跳出本次循环，而break跳出整个循环。
continue 语句用来告诉Python跳过当前循环的剩余语句，然后继续进行下一轮循环。
continue语句用在while和for循环中。

pass 语句
作用：是空函数，不做任何事情，用于暂时还未写逻辑和函数的里面

Python日期和时间
用函数方法
time.time( )来获取当前时间戳
格式化时间
print(time.strftime("%Y-%m-%d %H:%M:%S", time.localtime()))
