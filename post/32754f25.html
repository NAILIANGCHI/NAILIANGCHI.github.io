# 思路

让计算机随机从 0-100 取值，让用户输入数字来猜，用 if 语句来判断用户所输入的值与计算机的值相比较来输出大小结果。若值相等则程序结束反之，进入 while 循环继续让用户输入并判断至相同，每次判断都将 count 累加一来计算用户猜数的次数并在程序结束时告知用户 count 的值。
