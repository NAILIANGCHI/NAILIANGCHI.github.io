1+x web前端开发技术模拟试题

发表于2022-10-15|更新于2022-10-15|php
|阅读量:|评论数:

单项选择

1.php变量格式说法错误的是（ ）
A 变量以$开始
B 变量名中不能出现空格
C 变量名称不区分大小写
D 变量名不能以数字开始

正确答案：B

2.下列查询语法错误的是 ( )
A select * form table limit 10 offset 4;
B select * form table limits 10,5;
C select * from table limit 10,5;
D select * from table limit 10;

正确答案：B

3.php中关于包含文件上级目录的”a.php”的写法正确的是（ ）
A require “../a.php”;
B import ../a/php;
C require(“./a.php”)
D require ../a.php;

正确答案：A

4.下列数据类型中php5不支持的是( )
A Float
B int
C Boolean
D String

正确答案：B

5.下面不属于关键字let的特点的是( )
A 会产生变量提升现象
B 不能在函数内部重新声明参数
C 只在let命令所在的代码块内有效
D 同一个作用域，不能重复声明同一个变量

正确答案：A

6.下列是css预编译插件的是( )
A sasa
B css
C sass
D saas

正确答案：C

7.下列关于列位移描述错误的是( )
A col-pull-x 列向左移动x个单位
B col-offset-x 块元素向右移动x个像素
C col-pull-x 列向左移动x个单位
D col-offset-x 块元素向右移动x个单位

正确答案：B

8.PHP连接MySQL数据库步骤正确的是（ ）
1.执行sql语句 2.MySQL连接和数据库方法 3.设置字符集 4.选择数据库的方法 5.定义sql语句，并发送
A ②③④①⑤
B ③②④⑤①
C ②③④⑤①
D ①②③④⑤

正确答案：C

9.以下关于MySQL函数解释错误的是( )
A abc(x):返回x的绝对值
B trim(str):去除字符中的所有空格
C greatest(x1,x2,…,xn):返回集合中最大的值
D round(x,y):返回参数x的四舍五入的有y位小数的值

正确答案：B

10.下列PHP函数写法正确的是( )
A function void add( ){ }
B function add($x){return $x;}
C function add(x){return x;}
D function add($x){return $x;}

正确答案：B

11.MySQL的浮点数据类型不包括( )
A double
B decimal
C number
D float

正确答案：C

12.下列代码实现的效果是( )

A 显示堆叠在一起的两条不同颜色进度条
B 三条进度条不同长度，上下排列
C 显示一条进度条，颜色单一
D 显示堆叠在一起的三条不同颜色的进度条

正确答案：D

13.下列css命名规范正确的是( )
A 尽量使用类选择器进行样式选定
B 类命名时取父元素的class名作为前缀，使用_连接
C 严禁添加css代码注释
D 类命名样式之间以回车进行分割

正确答案：A

14.下拉列表中可以实现分割线的是( )
A dropdown
B disabled
C divider
D table

正确答案：C

15.下列关于PHP语法正确的是( )
A PHP文件可以包含HTML标签
B PHP语句以句号结束
C PHP脚本只能放在文档的开始位置
D <!-- -->是PHP中注释的符号

正确答案：A

16.以下哪个不是bootstrap的特点( )
A 包含大量的内置组件，易于定制
B 响应式设计
C 移动设备优先
D 闭源软件

正确答案：D

17.模糊查询姓张的人的查询语句书写正确的是( )
A select * form table where name like “%张%“
B select * from table like ‘张%’；
C select * form table where name like “张%”；
D select * form table where name like “_张“；

正确答案：C

18.按钮组大小类不含哪个( )
A btn-group-md
B btn-group-sm
C btn-group-xs
D btn-group-lg

正确答案：A

19.数据库的基本操作中写法错误的是( )
A update table set user=’user1’;
B insert into table (user,password)values(‘sys,’123456’);
C delete from table user=’user1’;
D select * form table where x>5;

正确答案：C

20.不是.container容器的自适应宽度为( )
A 320px
B 970px
C 750px
D 1170px

正确答案：A

21.下列警告框代码中显示为红色的是( )
A <div class="alert alert-success">成功</div>
B <div class="alert alert-info">信息</div>
C <div class="alert alert-danger">错误</div>
D <div class="alert alert-warning">警告</div>

正确答案：C

22.php类中变量默认的访问权限是( )
A public
B default
C private
D protected

正确答案：A

23.关于cookie与session说法错误的是( )
A cookie是在服务器端创建，并写回到客户端浏览器
B session将信息存在客户端，现在保存到服务器
C session保存的是对象，cookie保存的似乎字符串
D cookie对文件的大小要求控件在4k以内

正确答案：C

24.关于MySQL说法正确的是( )
A MySQL只能用于小型项目
B MySQL是收费的数据库
C MySQL是一种关系型数据库吗，以行列来存储数据库
D MySQL自身具备可视化界面

正确答案：C

25.以下对MySQL数据库操作错误的是( )
A use database_name;
B show databases;
C show table;
D drop database db_name

正确答案：C

26.查看数据表结构的语句是( )
A show tablename;
B show tables;
C desc tablename;
D show create tablename;

正确答案：C

27.HTTP协议中用于标识服务器内部错误的代码系列是( )
A 4XX
B 5XX
C 2XX
D 3XX

正确答案：B

28.bootstrap组件不包含( )
A 表格
B 轮播图
C 巨幕
D 分页

正确答案：B

29.Generator函数的yield关键字的作用是( )
A 停止执行
B 暂停执行，等待next( )方法调用
C 停止执行，可自行恢复执行
D 退出函数

正确答案：B

30.以下类可以实现斑马线效果的是( )
A table-striped
B table-jover
C table-bordered
D table

正确答案：A
